# Game-Script
Legendary 复制粘贴助手

一个基于 AutoHotkey v2 开发的自动化工具，可定时自动粘贴剪贴板内容并发送，适用于需要重复发送相同信息的场景。
✨ 功能特性

    定时自动发送：每6秒（可自定义）自动粘贴剪贴板内容并发送

    一键启停：PgUp 键快速启动/停止脚本

    智能提示：实时显示操作状态和反馈

    可视化设置：图形化界面调整发送间隔（1-60秒）

    统计信息：查看运行状态和发送记录

    托盘控制：系统托盘图标和右键菜单操作

    剪贴板保护：发送后自动恢复原始剪贴板内容

🚀 快速开始
系统要求

    Windows 操作系统

    AutoHotkey v2.0 或更高版本

安装步骤

    安装 AutoHotkey v2

    下载 Legendary复制黏贴助手.ahk 脚本文件

    双击运行脚本（或在AHK中打开）

🎮 使用方法
基本操作

    复制文本：先复制需要发送的文本到剪贴板

    启动脚本：按 PgUp 键开始自动发送

    停止脚本：再次按 PgUp 键停止发送

    退出脚本：按 F3 键完全退出程序
